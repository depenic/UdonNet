# UdonNet
My test for Neural net

# Install

First, cloning this repository
``git clone https://github.com/hdl-noodles/UdonNet.git``

Update the submodules
``git submodule update --init --recursive``

Install dependecies
``cd bindsnet_experiments``
``pip install .``
``cd ..``

Run the test script
``bash diehl_and_cook.sh``
